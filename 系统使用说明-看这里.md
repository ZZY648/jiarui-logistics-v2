## 各模块操作说明

登录后左侧是导航菜单，点击切换页面。右上角显示你的用户名和角色，旁边有退出按钮。

### 1. 数据看板（登录后默认页面）

**能看到什么：**
- 顶部 4 个数字卡片：今日订单数、运输中运单数、空闲/总车辆、本月完成数
- 运单状态分布：每种状态有多少个运单，用彩色卡片展示
- 近 7 天运输量柱状图：每天的发货数量一目了然
- 近 30 天客户发货 TOP 排行：哪个客户发货最多

**怎么操作：** 纯查看页面，点进去就能看到数据，不需要额外操作。

---

### 2. 运单管理

**能看到什么：** 最近 50 条运单的列表，包括运单号、客户名、货物名、报价金额、当前状态、创建时间。

**怎么操作：**
- 点击蓝色的**运单号**或**详情按钮**→ 弹出运单详情窗口
- 详情窗口里能看到：
  - 状态流转条（草稿→已确认→已调度→已装车→运输中→已到达→已签收→已完成），当前状态会高亮蓝色
  - 站点信息表（提货点、卸货点）
  - 费用明细表
  - 操作按钮（取决于当前状态）

**状态推进操作：**

运单状态会随你的操作自动流转，不同状态下可执行的操作如下：

| 当前状态 | 可以做的操作 | 点哪个按钮 |
|----------|-------------|------------|
| 已确认 | 派车调度 | 点 **派车(调度推荐)**，系统自动推荐最优车辆 |
| 已调度 | 确认装车 / 取消 / 标记异常 | 点 **确认装车** / **取消运单** / **标记异常** |
| 已装车 | 确认发车 / 标记异常 | 点 **确认发车** |
| 运输中 | 确认到达 / 标记异常 | 点 **确认到达** |
| 已到达 | 确认签收 | 点 **确认签收** |
| 已签收 | 完成归档 | 点 **完成归档** |

一个运单走完全流程：创建 → 确认 → 派车 → 装车 → 发车 → 到达 → 签收 → 归档。

### 3. 新建运单

**怎么填：**
1. **货物名称**（必填）：比如"电子元件 20箱"
2. **货物类型**（下拉选）：普通货物/易碎品/危险品/冷链
3. **重量(kg)**：填写公斤数
4. **体积(m³)**：填写立方米
5. **件数**：填写件数
6. **时效要求**（下拉选）：普通/加急/预约
7. **货物备注**（选填）：比如"易碎请轻放"
8. **提货联系人**：姓名 + 电话
9. **卸货联系人**：姓名 + 电话

**点「确认下单」**→ 系统自动生成运单号（格式：YD+日期+序号），报价自动计算。

成功后弹出提示，自动跳转到运单列表页，新运单显示在最上面。

---

### 4. 调度派车

**能看到什么：**
- 上半部分：等待调度的运单列表（状态为"已确认"的运单）
- 下半部分：所有车辆状态表（车牌、车型、状态、载重、容积）

**怎么操作：**
1. 在待调度运单那一行，下拉选择**车辆**和**司机**
2. 点右侧绿色 **派车** 按钮
3. 运单状态从"已确认"变为"已调度"

**注意：** 只有状态为"空闲(idle)"的车辆才会出现在下拉列表中。如果某辆车状态显示为"维修(maintenance)"则不可选。

---

### 5. 费用管理

**能看到什么：** 所有进行中运单的列表，每行有"记录费用"和"核算成本"按钮。

**记录费用：**
1. 点 **记录费用** → 弹窗
2. 选费用类型（油费/过路费/司机提成/装卸费/罚款/其他）
3. 填金额
4. 填说明
5. 点保存

**核算成本：**
1. 点 **核算成本** → 弹出成本快照
2. 显示：运费报价、直接成本、间接成本（折旧+保险分摊）、总成本、利润、利润率
3. 利润绿色为正，红色为亏本

---

### 6. 对账管理

**生成对账单：**
1. 选择客户（下拉选）
2. 选择年份和月份
3. 点 **生成对账单**
4. 系统自动汇总该客户当月所有已完成运单，生成一份对账单

**对账单流转：**
| 状态 | 含义 | 可操作 |
|------|------|--------|
| 草稿(draft) | 刚生成 | 点「发送客户」 |
| 已发送(sent) | 已发给客户 | 点「确认」 |
| 已确认(confirmed) | 客户确认无误 | 线下开具发票 |

**注意：** 生成对账单需要该客户在该月有状态为"已完成"且未入账的运单。

---

### 7. 客户管理

**能看到什么：** 5 家客户的信息表，包括编码、公司全称、简称、联系人、结算方式（月结/趟结/预付）、折扣率、地址。

**怎么操作：** 纯查看页面，了解客户基本信息。

---

### 8. 轨迹追踪

**怎么操作：**
1. 输入完整的运单号（比如 `YD202606140001`）
2. 点 **查询**
3. 显示：
   - 运单基本信息和状态
   - 运输路径（提货站→卸货站，带彩色竖线标识）
   - GPS 轨迹点表（时间、经纬度、速度）

**提示：** 新建运单时会自动生成几条模拟 GPS 轨迹点，方便测试。

---

## 测试建议

要完整测试运单全流程，推荐按以下顺序操作：

1. **新建运单** → 填表下单（状态：已确认）
2. **运单管理** → 找到刚创建的运单，点详情 → 点「派车(调度推荐)」（状态：已调度）
3. **运单管理** → 再点详情 → 点「确认装车」（状态：已装车）→ 点「确认发车」（状态：运输中）
4. **费用管理** → 记录几笔费用（油费、过路费等）→ 核算成本看利润
5. **运单管理** → 继续推进状态：确认到达 → 确认签收 → 完成归档（状态：已完成）
6. **对账管理** → 选择该客户和当月 → 生成对账单 → 发送 → 确认
7. **轨迹追踪** → 输入运单号查询 GPS 轨迹

## 常见问题

**Q: 页面显示不对，换了页面看不到数据？**
A: 检查左侧导航，当前页面对应的菜单项是高亮的（蓝色左边框+浅蓝底色）。

**Q: 某个按钮点了没反应？**
A: 可能你的角色没有权限。比如普通调度员不能操作对账功能。请用 admin 账号登录。

**Q: 系统关了怎么重新打开？**
A: 如果终端窗口关了，需要重新启动。双击桌面的 `启动系统.bat` 即可（如果没有，告诉我帮你创建）。
